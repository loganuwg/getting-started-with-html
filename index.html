HTML-Hypertext Markup Language

HTML Consists Of

1- What you see
2-What it looks like
3-Link to other pages

TAGS-

1. B Tag is used to make a letter or word BOLD.
2. Em Tag is used to italisize a word or letter.
3. Href Tag can put a Link into a web page.

4. Img Tag can add an image to a web page it has no closing tag.

SPACING-

You can add space in the text of your sentences by using the BR or P Tags.
BR Tag is an in line tag and drops the text underneath the previous text.
P Tag is a block tag and starts a new paragraph
